Quadratic Equations in One Unknown

For the quadratic equation ax 2 + bx + c = 0 , where a ≠ 0 :

(a) x = - b ± b 2 - 4 ac 2 a

(b) Discriminant Δ = b 2 - 4 ac

  • Δ > 0 2 roots
  • Δ = 0 1 root (2 equal roots)
  • Δ < 0 no roots
  • Δ ≥ 0 has root(s) (can be 1 or 2)

(c)

  • (i) Sum of roots = α + β = - b a
  • (ii) Product of roots = αβ = c a

Note: α 2 + β 2 = ( α + β ) 2 - 2 αβ

New equation: x 2 - ( α' + β' ) x + α'β' = 0

Graph

  • a > 0 : open upwards, minimum
  • a < 0 : open downwards, maximum
  • c = y-intercept = Sub ( 0 , y ) , that is x = 0 , find y

Vertex = ( - b 2 a , - b 2 - 4 ac 4 a ) = ( h , k )

If y = f ( x ) = a ( x - h ) 2 + k :

  • Axis of symmetry: x = - b 2 a [or x = h ]
  • Optimum/Maximum/Minimum = - b 2 - 4 ac 4 a [or k ]

Other:

  1. x -intercept = Sub ( x , 0 ) , that is y = 0 , find x
  2. Completing square: y = a x 2 + bx + c = a ( x + b 2 a ) 2 + c - b 2 4 a
  3. Important Skill: Write down all coordinates. Sub those points lying on the graph into the equations one by one.

Equation of Straight Lines

Find slope first.

    equation of the straight line: y - y 1 = m ( x - x 1 )
  2. (b) y = mx + c , m is the slope, c is the y-intercept
  3. (c) For an equation of the straight line Ax + By + C = 0 :
    • slope = - A B
    • y-intercept = - C B
    • x-intercept = - C A

Exponential and Logarithmic Functions

  1. (a) a m n = a m n , where m and n are integers and n > 0
  2. (b) Laws of indices:
    1. (i) a m × a n = a m + n
    2. (ii) a m ÷ a n = a m - n
    3. (iii) ( a m ) n = a mn
    4. (iv) ( ab ) m = a m b m
    5. (v) ( a b ) m = a m b m
    5 methods to solve:
    1. log x = log A → x = A
    2. A x = B → x log A = log B
    3. log A x = B → x = A B
    4. A x + 1 + A x = B → Factorize: A x ( A + 1 ) = B
    5. A 2 x + 1 + A x = B → Let y = A x : A y 2 + y = B
  3. (c) For M , N > 0 , a > 0 and a ≠ 1 :
    1. (i) log a ( MN ) = log a M + log a N
    2. (ii) log a ( M N ) = log a M - log a N
    3. (iii) log a ( M n ) = n log a M
    4. (iv) log a 1 = 0
    5. (v) log a a = 1
    [Change Base] log a b = log c b log c a = log b log a
  4. (d) L = log 10 ( I I 0 ) , where I is the intensity of sound and I 0 is the minimum audible sound intensity
  5. (e) log E = 4.8 + 1.5 R , where E is the energy released (in joules (J)) from an earthquake

More about Trigonometry

  1. (a) Trigonometric identities:
    1. (i) 90 ° , 270 ° changed: sin ↔ cos, cos ↔ sin, tan ↔ 1/tan
    2. (ii) 0 ° , 180 ° , 360 ° unchanged: sin ↔ sin, cos ↔ cos, tan ↔ tan
  2. (b)
    1. (i) area of △ ABC = 1 2 ab sin C
    2. (ii) Heron's Formula: area of △ ABC = s ( s - a ) ( s - b ) ( s - c ) , where s = 1 2 ( a + b + c )
    3. (iii) Sine Formula: a sin A = b sin B = c sin C
    4. (iv) Cosine Formula: c 2 = a 2 + b 2 - 2 ab cos C or cos C = a 2 + b 2 - c 2 2 ab
    a , b and c are the opposite sides of ∠ A , ∠ B and ∠ C respectively.

Permutation and Combination

  1. (a) The number of permutations of r objects from n distinct objects without repetition = P r n = n ! ( n - r ) ! [Consider Order]
  2. (b) The number of combinations of r objects from n distinct objects without repetition = C r n = P r n r ! = n ! r ! ( n - r ) ! [Not Consider Order]

More about Probability

  1. (a) If two events E and F are mutually exclusive, then P ( E or F ) = P ( E ) + P ( F )
  2. (b) For any two events E and F , P ( E or F ) = P ( E ) + P ( F ) - P ( E and F )
  3. (c) The complementary event of E is denoted by E ' . P ( E ' ) = 1 - P ( E )
  4. (d) If two events E and F are independent, then P ( E and F ) = P ( E ) × P ( F )
  5. (e) Conditional probability: For any two events E and F , P ( E and F ) = P ( E ) × P ( F after E has occurred ) , where P ( E ) > 0
